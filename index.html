BlinkRite

Active Project

A smart bias lighting system built to combat digital eye strain. BlinkRite monitors the user's blink rate in real time and automatically triggers bias lighting to encourage natural eye readjustment and reduce dry eyes.

Stack
Python / ML MediaPipe ESP32 · Embedded C++ Bluetooth KiCad · Custom PCB
Implementation
  • Used MediaPipe face-mesh landmarks to isolate the eye region, then trained a custom ML model on a dataset of 300 video samples to detect eye states locally via a PC webcam.
  • Wrote a Python script to process the video stream and transmit low-latency state changes (High / Normal / Low) over Bluetooth.
  • Developed embedded C++ firmware on an ESP32 to receive signals and control the lighting hardware.
Current Focus

Finalizing the custom PCB assembly (overvoltage/overcurrent protection, JST connectors), linking the hardware and software systems, and refining the ML model to accurately detect high-speed micro-blinks.

Automated Optical Inspection (AOI) System

Active Project

An automated optical measurement system for a high-volume CNC production line, replacing manual QA checks on electrical enclosures by verifying hole dimensions and locations to a 0.010" tolerance.

Stack
Python · OpenCV ESP32 · Custom PCB Optics & Polarization 3D Printing
Implementation
  • Sourced and integrated a high-resolution, distortion-free bare PCB camera.
  • Designed a custom 3D-printed camera housing that securely mounts a ring light, with polarized filters on both the lens and light source to eliminate glare from bare metal parts.
  • Engineered a custom ESP32-based PCB with a Time-of-Flight sensor to automatically detect inserted parts and trigger measurements, complete with status LEDs.
  • Developed Python/OpenCV software to extract dimensional overlays from an internal database and compare them against the physical part.
  • Implemented checkerboard camera calibration to correct lens distortion, with a known-size reference block for precise pixel-to-millimeter scaling.
